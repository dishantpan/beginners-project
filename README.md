## 🧠 Concepts Covered So Far

- `class` and `object` — blueprint vs real instance
- `private` — Encapsulation, data protection
- Constructor — automatic setup when object is created
- `this` keyword — field vs parameter disambiguation
- `int[]` arrays — clean data storage
- `(double)` casting — avoiding silent integer division bugs
- `System.out.printf("%.2f")` — professional formatted output
- Single Responsibility — one method does one job
- Git — version control, committing feature by feature
